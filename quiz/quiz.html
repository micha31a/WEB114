Questions

  1. Name a real-life situation where you would need to use an `if` statement.
  2. Taking a look at my bank account to know if I can afford something! If my bank account balance is at least greater or equal to the price of what I'm trying to buy, then I can purchase it. If it's not (else), then I can't.
  3. Explain the difference between `==` and `===`. Why is strict equality usually better?
  4. == will try to convert the values to the same type in order to check equality. However, === checks the type first and then the values if they're the same type to strictly check equality. If the types are different, then it won't proceed with checking any further and return false. Using === makes sure you haven't overlooked any types that are different in your code!
  5. Give an example (in your own words) of when you would use `&&` (AND) vs `||` (OR).
  6. If I want to make sure that all conditions are met in order to complete the next step, I use &&. If I only need one condition to be met, then I use ||.
  7. What does the `!` (NOT) operator do? Give an example of a situation you would use it.
  8. The NOT operator means that if the condition is met and NOT is used, it will flip the value to false instead of true. For example if I need exactly a certain amount of oranges, no more or less than that, I would use the NOT operator to cover all other situations where I don't have the exact number that I need.
